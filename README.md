# JumpTrack

JumpTrack is a command-line app that allows Track & Feild athletes to log their long jump and triple jump distances. It tracks your attempts, shows your personal bests, and shows your progress with a line chart.

## How to Run

1. Make sure to have the latest Python installed (3.10 or higher)
2. Install dependencies: pip install -r requirements.txt 
3. Run the app: python main.py

## Dependencies
- matplotlib

## How to use

When you start up the app you will be presented with 5 options.
1. Add a jump. - Log a new jump
2. View jumps. - View all the jumps thats been logged
3. View your personal best. - See your best jump for each event.
4. Exit. - Close the app
5. View progress chart - See your progress overtime on a chart.

Type in the number of which option you want to use and press enter
